# Fashion Footprint

A static website raising awareness about the environmental impact of the fashion industry.

## Pages
- **Home** (`webpage.html`) — About section, services, and contact form
- **Sustainable Brands** (`sustainable-brands.html`) — Indian eco-friendly fashion brands
- **NGOs** (`ngos.html`) — Organizations accepting clothing donations

## How to Run
Clone the repo and open `html/webpage.html` in your browser.

## Tech Stack
HTML · CSS · Font Awesome icons
